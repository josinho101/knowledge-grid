### How to run

##### Commands to install packages and start both server and client

- cd app
- cd server
- npm i
- cd ..
- cd client
- npm i
- cd ..
- npm i
- npm start

##### Commands to install packages and start server only

- cd app
- cd server
- npm i
- npm start

##### Commands to install packages and start client only

- cd app
- cd client
- npm i
- npm start
